# homebrew-tap

Homebrew tap for [pavelsimo](https://github.com/pavelsimo)'s CLI tools.

## Usage

```bash
brew tap pavelsimo/homebrew-tap
```

Then install any available formula:

```bash
brew install ani   # browse and search AniList anime from your terminal
```

## Formulas

| Formula | Description |
|---------|-------------|
| [ani](https://github.com/pavelsimo/ani) | Browse and search AniList anime from your terminal |
